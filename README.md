### Mow It Now

Mow it now is a simple mower simulator.

#### PreRequisite
Having both package installed
- Gradle 4.10.1+
- JDK 8+

#### Getting Started

##### How to run
```shell
git clone https://github.com/khantzen/mowitnow.git
cd mowitnow
gradle run --args='mower-simulator-entry'
```

##### How to build
```shell
gradle clean build
```
